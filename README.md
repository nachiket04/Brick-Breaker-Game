# Brick-Breaker-Game

This is a game in which using a slider we have to keep the ball bouncing upwards, so that all the bricks are destroyed. Its a desktop Java Application.
Softwares and Languages Used:
User Interface:

    Swing
    AWT

Back-end Logic:

    Java
